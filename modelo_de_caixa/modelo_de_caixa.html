Demonstrando o modelo de caixa

O modelo de caixa do CSS é essencialmente uma caixa que envolve cada elemento HTML. O modelo consiste de bordas, preenchimento, margens e do conteúdo propriamente dito.

Este texto é o conteúdo de uma caixa. Acrescentamos um preenchimento de 50px, uma margem de 20px e uma borda verde de 15px.
